Page elements

Plus button

At the bottom left corner of the page there is the plus button: it is needed to create groups, topics and proposals. When you hover the cursor on it a different number of buttons will popup, according to the current page:
  • Green button: lets you create a group, will appear in any screen
  • Blue button: lets you create a topic, will appear in the group screen and in the topic screen
  • Yellow button: lets you create a proposal, will appear in the topic screen

Actions

Create a group

To create a group:
  1. hover on the red plus button in the bottom left corner
  2. when the popup button show up, click the green one
  3. enter group information
  4. click the "Create" button

Create a topic

To create a topic:
  1. open the group in which you want to create the topic
  2. hover on the red plus button in the bottom left corner
  3. when the popup button show up, click the blue one
  4. enter topic information
  5. click the "Create" button

Create a proposal

To create a proposal:
  1. open the topic in which you want to create the proposal
  2. hover on the red plus button in the bottom left corner
  3. when the popup button show up, click the yellow one
  4. enter proposal information
  5. click the "Create" button
