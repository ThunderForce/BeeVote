BEEVOTE

Create your groups, propose in your topics and vote


This is an alpha version of BeeVote: only the basic features are present, including: Create groups, Add and remove group members, Create and remove topics, Create proposals, Vote proposals.

Availability of features may change, and data stored in the system might be wiped during the process of improving the application.

Using this application requires a registered Google Account. For now to register your Google Account in the application please follow these steps:
- Login with your Google Account
- Submit your name and surname
- Wait for an administrator to accept your registration request
